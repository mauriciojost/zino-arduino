# USER GUIDE

## BUTTONS

The module has 3 buttons for user interaction: 
- MODE BUTTON: to change between modes
- SET BUTTON: to change configuration values in a given mode
- RESET BUTTON: to reset the module

## MODES

- Hours
- Minutes
- Factor (+)
- Factor (-)
- Pump A
  - Amount
  - Test
  - Frequency
- Pump B
  - Amount
  - Test
  - Frequency
- Water Level
  - Minimum
  - Frequency
- Ready
- Run
